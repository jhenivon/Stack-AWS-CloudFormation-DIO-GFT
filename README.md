# 🚀 AWS CloudFormation - Firewall EC2 - DIO - GFT

## 🎯 Objetivo
Este laboratório tem como objetivo consolidar os conceitos de **AWS CloudFormation**, criando uma stack que provisiona uma instância EC2 com Apache e um **Security Group** atuando como firewall.  
A entrega é um repositório público no GitHub contendo documentação clara e organizada.

---

## 📚 Aprendizados
- O que é AWS CloudFormation e como funciona.  
- Diferença entre criar recursos manualmente e via template.  
- Estrutura básica de um arquivo YAML para stacks.  
- Criação de instâncias EC2 com configuração automática via **UserData**.  
- Configuração de **Security Groups** para controlar tráfego de rede (firewall).  
- Uso do GitHub como ferramenta de documentação técnica.
